Procedimento para a janela do DB prod

  1. Backup primeiro
    pg_dump -Fc -h … -U listaatendimento listaatendimento > omni-pre-rename.dump
  2. Derrubar serviços que conectam
    Parar containers api e web via docker compose stop para liberar conexões.
  3. Renomear o banco e o role
    ALTER DATABASE listaatendimento RENAME TO omni;
    ALTER ROLE listaatendimento RENAME TO omni;
  4. Atualizar .env de produção
    Trocar POSTGRES_DB, POSTGRES_USER, COMPOSE_PROJECT_NAME.
  5. Subir tudo
    docker compose -f docker-compose.prod.yml --env-file .env up -d
  6. Smoke check
    Login, criação de tenant, leitura de operação. docker compose exec api ./migrate up para confirmar idempotência.
